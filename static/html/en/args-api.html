API Documentation

Args exposes its search interface via a REST API which you can use in your own software to query search results programmatically.

API Basics

Access via: http://141.54.172.105:8081/api/v1/_search?query=GOOGLE

  • Use double quotes ("") for phrase queries
  • Use plus signs (+) between words to form multiple-word queries
  • Use ampersands (&) to add additional parameters to your query
  • The API will show the first 100 results by default. To specify please use our additional parameters "from" and "to"
  • Please note that the API will not output more than 100 arguments at all times

Additional parameters:

  • pretty
    • /!\For a readable output the &pretty parameter is highly recommended.
  • from=ARGUMENT_POSITION
    • ARGUMENT_POSITION needs to be a positive integer value
    • If not specified the default value will be 0
  • to=ARGUMENT_POSITION
    • ARGUMENT_POSITION needs to be a positive integer value
    • Default will be FROM_VALUE + 100
Example: http://141.54.172.105:8081/api/v1/_search?query=GOOGLE&pretty&from=5&to=15
